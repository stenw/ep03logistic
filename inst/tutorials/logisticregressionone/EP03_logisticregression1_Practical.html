Now we will look at the relation using logistic regression.

In R we can estimate a logistic regression model using the command glm()The first parameter indicates the model formula. On the left side of the ~ sign the dependent variable is specified (here agvhd); on the right hand side the explanatory variables (here the single variable sexmatch). The parameter family indicates the distribution we use. The default model for a binomial distribution is the logistic regression model. The last parameter specifies the data set we use.

bmt$sexmatch <- ifelse(bmt$sexrec==bmt$sexdon, 1, 0)
glm1<-glm(___ ~ ____, family = ____, data=___)
glm1

If the response is a 0 /1 coded variable R the model predicts the log odds of the outcome that is coded 1. If the response is a factor by default the first level is the refence level so we model the log odds of the last level.

# Look at the examples at the end of the chapter on logistic regression in the course notes and fill in the blanks.
# The code is:
glm1<-glm(agvhd ~ sexmatch, family = 'binomial', data=bmt)
glm1

Question 4

Use the summary command on the returned object to obtain the estimated coefficients standard errors and p-values from a Wald test. The odds ratios can be obtained by using exp(coef(glm1)). The function confint is used to obtain confidence estimates, again the exp function transforms them to the odds-ratio scale. Compare the OR with the result you obtained when you calculated it by hand.

bmt$sexmatch <- ifelse(bmt$sexrec==bmt$sexdon, 1, 0)
glm1<-glm(agvhd ~ sexmatch, family = 'binomial', data=bmt)
# The code is:
# summary(___)
# exp(coef(___))
# exp(confint(___))
# The code is:
summary(glm1)
exp(coef(glm1))
exp(confint(glm1))

The odds ratio should be the same than the one you computed manually. The odds ratio is approximately a half indicating that we expect the odds of agvhd to be halved when the sex of the host matches that of the donor. The effect is not statistically significant at the 5% level according to the Wald test. Also 1 is included in the 95% confidence interval. (Usually the conclusion based on the confidence interval agrees with that of the Wald test but this is not always the case as the methods on which they are based are different.)
